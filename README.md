md2wechat 是专为 AI 时代设计的公众号创作工作台：写 Markdown，一键转换为精美排版并推送微信草稿箱，支持 43 个高级排版模块、40+ 专业主题和 AI 配图。

本仓库是完整的使用教程合集，覆盖从安装到 API 接入的全流程。

## 快速上手

新手推荐从这里开始 👇

**5 分钟最小路径：**

```bash
# 1. 安装
brew install geekjourneyx/tap/md2wechat

# 2. 配置微信凭证（只需一次）
md2wechat config init

# 3. 转换第一篇文章
md2wechat convert article.md --draft
```

→ 完整说明见 [快速上手指南](./01-quick-start.md)
